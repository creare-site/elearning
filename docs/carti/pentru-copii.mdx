# Pentru Copii

Iata cinci recomandari de carti pentru copii:

## "Micul Print" de `Antoine de Saint-ExupÃ©ry`

O carte minunata despre prietenie, iubire si importanta lucrurilor simple din viata. Este potrivita pentru copiii mai mari si poate fi o lectura emotionanta si inspiratoare pentru adulti.

<br />

## "Pisica in Cizme" de `Charles Perrault`

O poveste clasica despre aventurile unui motan istet care isi ajuta stapanul sa devina bogat si sa se casatoreasca cu o printesa. Este o carte amuzanta si plina de invataminte pentru copiii mici.

<br />

## "Hobbitul" de `J.R.R. Tolkien`

O carte fantastica despre o calatorie epica prin taramurile imaginare ale Terrei de Mijloc, populata de personaje fantastice precum hobbiti, elfi, pitici si dragoni. 
Este potrivita pentru copiii mai mari si poate fi o lectura captivanta si educativa.

<br />

## "Frumoasa si Bestia" de `Gabrielle-Suzanne Barbot` 

O poveste romantica despre o tanara frumoasa care se indragosteste de o fiara infricosatoare, care se dovedeste a fi un print fermecat. Este o carte clasica si inspiratoare pentru copiii mici.

<br />

## "Harry Potter" de `J.K. Rowling` 

O carte despre aventurile unui baiat orfan care descopera ca este un vrajitor si incepe sa invete magia la o scoala secreta pentru vrajitori. 
Este potrivita pentru copiii mai mari si poate fi o lectura captivanta si educativa despre prietenie, curaj si lupta impotriva raului.
